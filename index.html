git Memento

Une sélection de commande pour git

Branches
Lister les branches locales git branch
Créer une branche et switcher dessus git checkout -b (nom-branche)
Changer le nom d'une branche git branch -m (ancien-nom) (nouveau-nom)
Switcher sur une branche existante git checkout (nom-branche)
Renommer la branche courante git branch -m (ancien nom) (nouveau-nom)
Supprimer une branche en local git branch -d (nom-branche)
Supprimer une branche en local peut importe si elle a été mergé ou pas git branch -D (nom-branche)
Commit
Corriger une erreur dans un message de commit git commit --amend -m (Nouveau message)
Ajout
Ajouter les fichiers au fur et à mesure git add -p
Ajouter tous les fichiers avec l'extension .html git add *.html
Logs
Historique des commits sur une seule ligne git log --oneline
Historique des 2 derniers commits git log -2
Historique des 2 derniers commits avec les diff. git log -p -2
Toutes les modifications en locales git diff
Modification sur un fichier git diff (nom fichier)
Blamer
Obtenir des informations sur un fichier git blame (nom du fichier)
Afficher les informations des lignes 10 à 20 git blame -L 10,20 (nom du fichier)
Nettoyer
Supprimez tous les fichiers non suivis git clean -f
Supprimez tous les fichiers et répertoires non suivis git clean -df
Historique des 2 derniers commits avec les diff. git log -p -2
Toutes les modifications en locales git diff
Modification sur un fichier git diff (nom fichier)
Annuler toutes les modifications locales git checkout -- .
Retirer un fichier suivi git reset HEAD (nom fichier)
Stashes
Sauvegarder les modifications + message git stash save (message) && git stash
Liste des stashes git stash list
Aide
Obtenir des informations sur une commande git help (commande)
